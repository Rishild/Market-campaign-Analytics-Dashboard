Marketing Campaign Analytics Dashboard

This repository contains visual dashboards that provide a comprehensive overview of marketing campaign performance, user engagement, and ad analytics across multiple demographics, interests, and platforms.
The dashboards were built to support data-driven decision-making for campaign optimization, budget allocation, and audience targeting.
